# MediClear Frontend

## Setup

```bash
cd mediclear-frontend
npm install
```

## Run

```bash
npm run dev
```

Open `http://localhost:5173`. Click "Check backend connection" — make sure the
backend is running first (`npm run dev` inside `mediclear-backend`). You should
see: "MediClear backend is running".

This confirms Milestone 1's frontend-backend connection is working.
